# Parcel Delivery Web Frontend

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create a `.env` file (optional):
```
VITE_API_URL=http://localhost:5000/api
```

3. Start the development server:
```bash
npm run dev
```

The app will be available at http://localhost:3000

## Features

- Admin Dashboard: Manage users, categories, services, and view all bookings
- Customer Dashboard: Browse services, create bookings with map location picker, track bookings
- Provider Dashboard: View and accept pending jobs, update job status

## Default Login

- Admin: admin@example.com / admin123
- Register new users through the registration page
